Introducion

Spritify is a easily library to manage yours sprites, use the spritify is easy, fast and standalone, they uses no plugin. look the demo


Demo



How use

Is so simple to use Spritify. see the steps

  • Download the file spritify.js here !
  • import the code on your HTML file
    <script type="text/javascript" src="spritify.js"></script>
  • import your CSS file with parameters of sprite
    <link rel="stylesheet" href="path/of/sprite.css"></link>
  • and implement
    sprites.animate({
        frames : 6,
        loop : false,
        target : "#sprite",
        animationTime : 70,
        spriteClass : "icon-sprite",
        reverse : false
    });

What can i use ?

frames: int //set the max of frames your sprite have. *this is required to work
loop: boolean //set if your sprite have loop or not *default = true
target: "selector" //set the element that will receive the animation *this is required to work
animationTime: int //set the frame time of animation *default = 70ms
spriteClass: "class" //set the class prefix of animation *this is required to work **example: "anim-1", "anim-2" in this case the class prefix is "anim-"
reverse: false //set if the animation is backwards *default = false
firstNumber: int //set the first frame of animation *default = 1
